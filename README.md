# Aplikacje Mobilne - projekt 2023/2024

### How to run it

> *optional*
> 
> `python -m venv venv && source venv/bin/activate`


1. Install required requirements
```bash
pip install -r requirements.txt
```
2. Run the app
```bash
cd app
python main.py
```

### How to run it (docker version)

```
docker-compose up
```
